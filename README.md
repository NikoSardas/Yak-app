## Description
An app to provide users with a chat interface and options to share images and their
location.

#### Built with
- React Native
- Expo
- Firebase (requires an account)
- Gifted Chat

### CLI install and run
```bash
git clone https://github.com/NikoSardas/Yak-app.git
cd yak-app/
```

```bash
npm install --global expo-cli
npm install
```

```bash
expo start
```
